# RuchiJS
It's a javascript library for our utilities &amp; we're making it better day by day. 

# Instalation
1. Using node package manager:
    ```shell
    npm i ruchijs
    ```
2. Using yarn
    ```shell
    yarn add ruchijs
    ```
